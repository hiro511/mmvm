# mmvm

## Usage

### Build
  `$ make`

### Run
  `$ bin/mmvm a.out`

### Options

| options | description |
| --- | --- |
| -m | display register values per an instruction |
| -d | run as disassembler |

## Directory

- mmvm2

  code base

  - include

    header files

- testdata

  test data

- bin

  executable file
